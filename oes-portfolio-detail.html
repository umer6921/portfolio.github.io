Online Examination System

Project information

  • Category: Web Application
  • Client: Final Year Project
  • Project Description: This application is developed using HTML, CSS, Boostrap, JavaScript and Php. It would make it possible for educational institutions to run examinations, quizzes, and feedback forms.
    It requests that instructors develop their own set of questions. The instructor of the question set would have access to the response's outcome.Additionally, the student would get the results through mail.
